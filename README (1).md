- 수집한 데이터 파일에서 모든 컬럼이 유의미하지 않아도 됨

- 기본 제공되는 데이터 3종을 파악

- feature engineering 아이디어 구상(ex. a컬럼과 b컬럼을 곱한 값을 추가 or 이전 몇개의 c컬럼의 값들을 조합한 값을 추가 등)
